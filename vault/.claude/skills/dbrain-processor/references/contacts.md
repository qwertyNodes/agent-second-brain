# Контакты

## Протокол поиска

При поиске человека или контакта:
1. Поиск по vault CRM-файлам по имени
2. Проверка daily-файлов на недавние упоминания
3. Поиск по ассоциации с компанией/проектом

## Распознавание имён в entries

Ищи паттерны:
- «Стас написал...», «Стас проверил...» → Команда FiveBBC
- «Дима добавил...», «Дима тестирует...» → Команда FiveBBC
- «оптовик», «клиент написал» → Оптовый клиент (создать задачу)
